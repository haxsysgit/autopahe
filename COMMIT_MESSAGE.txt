Migration Guide:
================
No migration needed! All existing commands work unchanged.
New features are additive and optional.

To use new features:
1. Generate config: python3 auto_pahe.py --write-config
2. Try filters: python3 auto_pahe.py -s "Naruto" --year 2007
3. Enable notifications: python3 auto_pahe.py -s "Naruto" -i 0 -d 1 --notify
4. Manage records: python3 auto_pahe.py -R view
